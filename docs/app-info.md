# 应用说明
**Device Detector**: 基于flutter的网络设备扫描器

## 功能
- 根据本机ip地址或输入的ip范围扫描网络设备
- 展示可访问的ip地址、延迟时间、设备mac地址和制造厂商

## 支持的平台
- android(armv7,armv8)
- windows(x64)
- linux(x64)
